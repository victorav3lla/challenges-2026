### Agregar más recorridos

¿Terminaste con tu Core User Journey, prototipo y preparación del pitch? ¡Por qué no construir otro recorrido de usuario en tu prototipo!

Hasta ahora, nos enfocamos exclusivamente en el recorrido **más importante** de tu producto: el Core User Journey. Pero el producto que construirás durante las últimas semanas del bootcamp puede tener más de un recorrido.

Con el ejemplo de Airbnb, recorridos adicionales podrían ser:
- Cancelar / Actualizar una reserva
- Dejar una reseña

O incluso recorridos desde la perspectiva de otro *tipo de usuario*, el propietario:
- Crear un nuevo anuncio
- Actualizar el precio por noche
- Agregar fotos a un anuncio existente
