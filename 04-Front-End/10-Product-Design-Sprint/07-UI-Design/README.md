## UI Design (30mn)

Time to add the finishing touch to your prototype in the form of Visual Design!

You can adjust several aspects to make your prototype look as close as possible to an actual web application:
- Images
- Fonts
- Colors
- Icons
- Animations

### Adding Images

There are two main ways to add images in your Figma design:
- Use the Unsplash plug-in

![Add Images with Unsplash](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/dx60ylrisk12qew5rzxhver6r77r)

- Manually download and add an image

You can even edit several images at the same time!

![Edit Several Images](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/96xt11f7mqymprmeum3bwq5pnecl)

### Adding Icons

Now that you are front-end masters, you know the best resource to download free icons: [FontAwesome](https://fontawesome.com) is your best friend here as well.
Download an icon of your choice, drag and drop it into Figma and voilà!

You can also find and add icons directly from the Design System.

![Add icons from Design System](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/acwa88e49x2ximbcbduyk3a6go7m)

### Adjusting an image contrast with a gradient

In some cases, images have high contrast, which can make the text above it hard to read. A great designer trick is to add a gradient on top of that image to enhance visbility.

![Adding Gradient](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/n74jtxuuiibv608i5a7dy8pswm7a)

### Fonts & Colors

Using Figma's Simple Design System has some limitations, and updating Fonts and Colors is one. It is possible to do it, but you lose the benefit of a Design System.

For today, we recommend:
- A maximum of 2 or 3 font families
- A maximum of 2 or 3 different colors, used sparsely

If you wish to update a specific font, you will need to "detach" it from the Design System. Note that this is a workaround **for today only**.

![Update Font](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/b7p0baosjrifnpcpecwuq04kuakj)

Similarly, if you wish to use specific colors for buttons or other components, you will need to "detach" it from the Design System.

![Update Button Color](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/le2jul8i7gkjp4wqnmzb7z0x4rk4)

**Pro Design Tip:** Use fonts and colors with intent, to attract your user's attention to a given action for example!
