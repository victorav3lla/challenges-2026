## UI 设计 (30分钟)

是时候以视觉设计的形式为你的原型添加点睛之笔了！

你可以调整多个方面，使你的原型尽可能接近真实的 web 应用程序：
- 图片
- 字体
- 颜色
- 图标

### 添加图片

在 Figma 设计中添加图片有两种主要方式：
- 使用 Unsplash 插件

![使用 Unsplash 添加图片](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/dx60ylrisk12qew5rzxhver6r77r)

- 手动下载并添加图片

你甚至可以同时编辑多张图片！

![编辑多张图片](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/96xt11f7mqymprmeum3bwq5pnecl)

### 添加图标

现在你是前端大师了，你知道下载免费图标的最佳资源：[FontAwesome](https://fontawesome.com) 在这里也是你的好朋友。
下载你选择的图标，拖放到 Figma 中，搞定！

你也可以直接从设计系统中找到并添加图标。

![从设计系统添加图标](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/acwa88e49x2ximbcbduyk3a6go7m)

### 用渐变调整图片对比度

在某些情况下，图片对比度很高，这可能会使上面的文字难以阅读。一个很棒的设计师技巧是在图片上添加渐变以增强可见性。

![添加渐变](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/n74jtxuuiibv608i5a7dy8pswm7a)

### 字体和颜色

使用 Figma 的简单设计系统有一些限制，更新字体和颜色就是其中之一。这是可以做到的，但你会失去设计系统的好处。

对于今天，我们建议：
- 最多 2 到 3 种字体系列
- 最多 2 到 3 种不同的颜色，谨慎使用

如果你想更新特定的字体，你需要将其从设计系统中"分离"。请注意，这是**仅限今天**的变通方法。

![更新字体](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/b7p0baosjrifnpcpecwuq04kuakj)

同样，如果你想为按钮或其他组件使用特定颜色，你需要将其从设计系统中"分离"。

![更新按钮颜色](https://wagon-public-assets.s3.eu-west-3.amazonaws.com/le2jul8i7gkjp4wqnmzb7z0x4rk4)

**专业设计提示：** 有目的地使用字体和颜色，例如吸引用户注意特定的操作！
